# Sublime-text 

Configuracion Local de Sublime text 4  +  LSPs  + Linter + VSCode Shortcuts

Instalacion: 
La carpeta `.config` se debe copiarse a `$HOME`:
`cp -r .config $HOME`

Configuracion Incluida: 
 - Babel
 - `LSP - jdtls`(JAVA)
 - `LSP - typescript`
 - SublimeLinter.
 - VSCode key binding + mouse binding.
 - Custom Context Menu.
